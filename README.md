# 주요 기능 
## 1. 회원 기능 🐹🤍🐹
- 회원가입
- 로그인
- 회원목록
- 회원조회
- 회원정보 수정
- 회원삭제
- 로그아웃
  
## 2. 게시판 기능 🐹📝
- 게시물 작성
- 게시물 수정 및 삭제
- 게시물 좋아요 및 좋아요 취소
- 베스트 게시물
- 댓글 작성
- 댓글 수정 및 삭제
- 댓글 좋아요 및 좋아요 취소
- 베스트 댓글
- 게시판 분할
